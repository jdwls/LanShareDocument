# 局域网实时协作Markdown文档系统

基于Flask和Flask-SocketIO构建的局域网实时协作Markdown文档系统，专为教室环境设计。

## 功能特性

### 教师端功能
1. **教师登录** - 使用预设密码登录教师控制台
2. **上传学生分组Excel** - 上传包含班级、组别、姓名的Excel文件
3. **班级选择与激活** - 选择当前上课的班级，学生只能登录被激活的班级
4. **查看小组状态** - 查看各小组在线成员和文档状态
5. **重置文档内容** - 教师可以重置指定小组的文档内容

### 学生端功能
1. **学生登录** - 通过姓名登录，系统自动分配对应小组
2. **组内实时协作编辑** - 同组成员实时同步编辑Markdown文档
3. **成员状态感知** - 显示在线成员列表，实时更新成员状态
4. **Markdown实时预览** - 支持Markdown语法，实时预览编辑效果

### 实时协作机制
- 基于Socket.IO的WebSocket通信
- 每个小组对应一个Socket.IO房间
- 文档内容实时同步，无需保存
- 在线成员状态实时更新

## 系统要求

- Python 3.7+
- Flask 2.3.3
- Flask-SocketIO 5.3.6
- pandas 2.0.3
- openpyxl 3.1.2
- eventlet 0.33.3

## 安装与运行

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行系统
```bash
python app.py
```

### 3. 访问系统
- 系统启动后访问: http://127.0.0.1:5000
- 教师登录: http://127.0.0.1:5000/teacher/login
  - 默认密码: `teacher123`
- 学生登录: http:///127.0.0.1:5000/student/login

## 使用说明

### 教师操作流程
1. 使用密码 `teacher123` 登录教师控制台
2. 上传学生分组Excel文件（格式要求见下文）
3. 选择当前上课的班级并激活
4. 学生即可登录对应班级

### 学生操作流程
1. 访问学生登录页面
2. 输入姓名（需与Excel文件中的姓名一致）
3. 系统自动分配小组，进入协作编辑器
4. 开始实时协作编辑Markdown文档

### Excel文件格式要求
文件必须包含三列，列名必须完全一致：
- **班级** - 班级名称
- **组别** - 小组名称
- **姓名** - 学生姓名

示例格式：
```csv
班级,组别,姓名
计算机1班,第1组,张三
计算机1班,第1组,李四
计算机1班,第2组,王五
```

## 技术架构

### 后端技术栈
- **Flask** - Web框架
- **Flask-SocketIO** - WebSocket实时通信
- **pandas** - Excel文件解析
- **eventlet** - WebSocket服务器

### 前端技术栈
- **HTML5/CSS3** - 页面结构和样式
- **JavaScript** - 交互逻辑
- **Socket.IO Client** - 实时通信客户端
- **CodeMirror** - Markdown编辑器
- **marked.js** - Markdown解析器
- **Font Awesome** - 图标库

### 数据存储
- **groups.json** - 学生分组数据（持久化）
- **documents.json** - 文档内容数据（可选持久化）
- **Session** - 用户登录状态
- **内存缓存** - 实时文档内容（主要存储）

## 实时协作原理

1. **房间机制** - 每个小组对应一个Socket.IO房间
2. **事件驱动** - 编辑事件通过WebSocket实时广播
3. **内容同步** - 防抖机制避免频繁同步
4. **状态管理** - 在线用户列表实时更新

### DeepSeek AI功能
系统集成了DeepSeek AI助手，支持以下特性：

1. **API配置方式**：
   - API配置从前端获取，不保存到服务器
   - 教师端配置页面仅用于显示和测试
   - 学生端AI助手需要手动输入API配置

2. **配置参数**：
   - API密钥（必需）
   - API地址（可选，默认：https://api.deepseek.com/v1/chat/completions）
   - 模型名称（可选，默认：deepseek-chat）

3. **使用场景**：
   - 学生可以使用AI生成文档内容（仅组长可用）
   - 教师可以测试API连接
   - 支持重试机制和错误处理

## 故障排除

### 常见问题
1. **无法上传Excel文件**
   - 检查文件格式是否为.xlsx或.xls
   - 检查文件是否包含"班级"、"组别"、"姓名"三列
   - 检查文件大小是否超过16MB

2. **学生无法登录**
   - 检查教师是否已激活班级
   - 检查学生姓名是否与Excel文件一致
   - 检查网络连接是否正常

3. **实时同步失效**
   - 检查WebSocket连接状态
   - 检查防火墙设置是否允许WebSocket
   - 重启应用服务器
